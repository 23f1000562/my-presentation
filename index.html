Financial Data Processing Pipeline


import pandas as pd

# Load quarterly dataset
df = pd.read_csv("earnings.csv")

# Calculate Net Margin
df["net_margin"] = df["net_income"] / df["revenue"]

# Aggregate performance by business unit
summary = (
    df.groupby("business_unit")["net_margin"]
      .mean()
      .sort_values(ascending=False)
)

Discounted Cash Flow (DCF)

DCF valuation formula:

\[ PV = \frac{CF}{(1 + r)^n} \]

Where:

\[ CF = \text{future cash flow}, \quad r = \text{discount rate}, \quad n = \text{periods} \]
